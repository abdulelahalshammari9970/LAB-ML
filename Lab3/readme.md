
## Lab3 Overview
This project performs an Exploratory Data Analysis (EDA) on the US Stores Sales dataset. The dataset contains sales records from American stores between 2010 and 2011. The main goal is to understand the data, detect patterns, relationships, and trends that can help in making data-driven decisions.

---
## Dataset Description
The dataset is sourced from Kaggle: US Stores Sales.
It includes 20 columns and 4248 rows of sales transactions.

Key Columns:
Sales: Total sales value in dollars.
Profit: Profit earned.
Marketing: Marketing expenses.
Inventory: Inventory value at the time of sale.
COGS: Cost of Goods Sold.
State, Market, Market Size: Geographic and market information.
Product, Product Type, Type: Product details.
Date: Date of sale.

1. Data Loading and Initial Inspection
2. Data Cleaning

4. Univariate Analysis
5. Bivariate Analysis
